课程：编译原理

一、语言概述

语言的特征

  • 自然语言：语义难以形式化（如“汉字的序顺并不定一能影阅响读”）
  • 计算机语言：严格的语法和语义，易于形式化

形式化描述层次

层次 含义 规则
语言 满足一定条件的句子集合 语法规则
句子 满足一定规则的单词序列 语法规则
单词 满足一定规则的字符串 词法规则

程序设计语言的形式化

  • 程序 = 语句序列（满足语法规则）
  • 语句 = 单词序列（满足语法规则）
  • 单词 = 字符串（满足词法规则）

示例：

  • 赋值语句：变量 := 表达式
  • 条件语句：if 条件 then 语句
  • 循环语句：while 条件 do 语句

形式语言与自动机理论

  • Chomsky（乔姆斯基）：从生成语言的角度研究
  • Kleene（克林）：创立自动机理论
  • 1959年：Chomsky 将两者结合，证明文法与自动机的等价性
  • BNF范式：成功描述 ALGOL-60，实质就是上下文无关文法

二、基本定义

2.1 字母表与字符串

概念 定义 示例
字母表 Σ 非空有穷集合 {0,1}、{a,b,c}
乘积 Σ₁Σ₂ {ab | a∈Σ₁, b∈Σ₂} {0,1}{a,b} = {0a,0b,1a,1b}
n次幂 Σⁿ Σⁿ = Σⁿ⁻¹Σ {0,1}³ = {000,001,…,111}
正闭包 Σ⁺ Σ ∪ Σ² ∪ Σ³ ∪ … 至少一个字符
克林闭包 Σ* Σ⁰ ∪ Σ⁺ 包含空串 ε

2.2 字符串的基本概念

  • 长度 |x|：字符串中字符的个数，|ε| = 0
  • 并置 xy：x 直接连接 y
  • 前缀/后缀：y 是 x 的前缀如果 ∃z 使 x=yz
  • 子串：w 是 x 的子串如果 ∃u,v 使 x=uwv

2.3 语言的定义

定义：L ⊆ Σ*，L 称为字母表 Σ 上的语言，∀x∈L，x 叫做 L 的一个句子。

示例：Σ={0,1} 上的语言

  • {0,1}
  • {00,11}
  • {0,1,00,11}*
  • {01,10}*

三、文法的定义

3.1 文法 G 的形式定义

G = (V, T, P, S)

符号 含义 说明
V 非终结符集 语法变量，代表语言子结构
T 终结符集 句子中出现的字符，V ∩ T = ∅
P 产生式集 α→β，定义语法结构
S 开始符号 S∈V，代表文法定义的语言

3.2 产生式

  • 左部 α：至少包含一个非终结符
  • 右部 β：终结符和非终结符的串（可为空）
  • 简写：α→β₁|β₂|...|βₙ

3.3 赋值语句文法示例

G1: A → B = E
B → C | D
C → a | b | c
D → m[1] | m[2] | m[3]
E → COC | COD | DOC | DOD
O → + | -

四、推导与归约

4.1 基本概念

操作 方向 含义
推导 ⇒ 产生语言 α→β，γαδ ⇒ γβδ
归约 ⇐ 识别语言 α→β，γβδ ⇐ γαδ

4.2 多步推导

  • ⇒ⁿ：恰好 n 步
  • ⇒⁺：至少一步
  • ⇒*：零步或多步

4.3 句型与句子

概念 定义 是否含非终结符
句型 S ⇒* α, α∈(V∪T)* 可能含有
句子 S ⇒* w, w∈T* 不含

4.4 推导示例

证明 a = b + m[1] 是文法 G1 的句子：
A ⇒ B = E
⇒ C = E
⇒ a = E
⇒ a = C O D
⇒ a = b O D
⇒ a = b + D
⇒ a = b + m[1]

五、文法的分类（Chomsky体系）

5.1 四种文法对比

类型 名称 产生式限制 语言
0型 短语结构文法(PSG) 无限制 PSL
1型 上下文有关(CSG) |β| ≥ |α| CSL
2型 上下文无关(CFG) α∈V CFL
3型 正规文法(RG) A→aB 或 A→a（右线性）
A→Ba 或 A→a（左线性）		 RL

5.2 包含关系

RG ⊂ CFG ⊂ CSG ⊂ PSG
RL ⊂ CFL ⊂ CSL ⊂ PSL

5.3 示例判断

文法 产生式 类型
G6 S→ε | 0S 正则文法
G7 S→ε | 00S111 上下文无关
G9 含 0A→A0 等 上下文有关

六、CFG的语法树

6.1 语法树的构成

  • 树根：开始符号
  • 中间结点：非终结符
  • 叶结点：终结符或非终结符

6.2 示例

文法：E → E+E | E*E | (E) | id

句子：id + id * id 的语法树
E
/|
E + E
| /|
id E * E
| |
id id

6.3 短语、直接短语、句柄

概念 定义
短语 子树的所有叶子从左到右排列
直接短语 仅有父子两代的子树的所有叶子
句柄 最左直接短语

七、最左推导与最右推导

推导类型 策略 对应归约
最左推导 每次替换最左变量 最右归约
最右推导 每次替换最右变量 最左归约（规范归约）

最左推导示例

E ⇒ E+E ⇒ id+E ⇒ id+EE ⇒ id+idE ⇒ id+id*id

最右推导示例

E ⇒ E+E ⇒ E+EE ⇒ E+Eid ⇒ E+idid ⇒ id+idid

八、文法的二义性

8.1 定义

  • 如果一个文法的某个句子存在两棵不同的语法分析树，则该文法是二义性的
  • 二义性问题是不可判定的（不存在算法能判定任意CFG是否无二义性）

8.2 二义性示例

文法：E → E+E | E*E | (E) | id

句子：id + id * id 存在两种不同的推导，对应两棵不同的语法树。

8.3 消除二义性

通过引入优先级和结合律：

E → E + T | T
T → T * F | F
F → id | (E)

8.4 if语句的二义性

二义性文法：

S → if expr then S
| if expr then S else S
| other

二义性句子：if e1 then if e2 then s1 else s2

实践中，在能驾驭的情况下可以使用二义性文法，并附加优先级规则。

九、本章小结

  • 语言 = 句子集合，文法是有穷描述无穷语言的手段
  • 文法 G = (V, T, P, S)
  • Chomsky 分类：0型→1型→2型→3型（限制递增）
  • CFG 的语法树直观表示推导过程
  • 短语、直接短语、句柄是归约的核心概念
  • 二义性不可判定，但可通过改写文法消除
